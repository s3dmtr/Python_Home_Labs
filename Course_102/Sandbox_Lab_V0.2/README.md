# Sandbox Lab V0.2 - Secure Sample Registry

## Overview

This lab is the second practical milestone in building the **Secure Cloud Sandbox V2** graduation project.

The goal is to apply the concepts learned in **Python 102 (Tuwaiq Academy)** by creating a simple sample registry system that manages malware sample metadata.

---

## Python Concepts Used

- Lists
- Dictionaries
- Functions
- Date & Time (`datetime`)
- Random (`random`)
- Loops
- Conditions
- Input Validation

---

## Features

- Add a new sample
- Automatically generate a fake SHA-256 hash
- Search by file name
- Search by hash
- Search by threat type
- Display all samples
- Show statistics
- Find oldest sample
- Find newest sample
- Sort samples by upload date
- Reverse sample order
- Calculate days since upload
